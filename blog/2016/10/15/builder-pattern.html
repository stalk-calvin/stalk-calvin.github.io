Builder Pattern vs Factory Method Pattern


15 Oct 2016 Comments
#builder #design #pattern

Builder? Factory? Pattern? huh?

Builder pattern is one of object creational software design pattern. This pattern is often compared with “Factory” method pattern because factory method is also object creational design pattern. The key difference is how the object is created though. For example, createCar might return Honda or Audi. Now with the Builder pattern you could add colours to such objects, like, 4 all wheel drive red Honda with v6 engine etc. Basically, factory method pattern would create an object via createCar. Builder pattern has ability to add granularity to the object by applying chains of function calls.
